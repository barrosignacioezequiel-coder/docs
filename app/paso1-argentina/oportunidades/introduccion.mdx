# Oportunidades por rubro — Fábricas argentinas

> **Estado:** Esta sección se irá completando progresivamente con investigación de mercado y fábricas reales.

El objetivo de esta sección es documentar las **mejores oportunidades** para comprar productos directamente de fábricas argentinas y revenderlos, evaluando cada rubro con criterios objetivos y cada fábrica con una ficha detallada.

## Cómo se construye esta sección

<Steps>
<Step title="Análisis de mercado">
Se investiga el mercado argentino para identificar qué rubros tienen mayor demanda, mejores márgenes y menor competencia para un revendedor que empieza.
</Step>
<Step title="Selección de rubros">
Se seleccionan los 6-8 rubros con mayor potencial según 9 criterios ponderados (margen, demanda, capital, regulación, competencia, logística, accesibilidad, escalabilidad, puente a importación).
</Step>
<Step title="Búsqueda de fábricas">
Para cada rubro seleccionado, se buscan fábricas reales usando todas las fuentes documentadas en la sección [Encontrar Fábricas](/app/paso1-argentina/encontrar-fabricas/introduccion).
</Step>
<Step title="Documentación">
Cada fábrica encontrada se documenta con una **ficha individual completa**: datos de contacto, productos, precios estimados, pedido mínimo, scoring de oportunidad.
</Step>
</Steps>
