# Cap. 73 — Manufacturas de hierro o acero (Otoño)

> **Capítulo NCM 73** — Manufacturas de fundición, hierro o acero | **Temporada:** Otoño (Mar–May)

El otoño trae dos demandas específicas: **percheros reforzados** para abrigos pesados (camperas, tapados, bufandas) y **bombillas de acero inoxidable** para mate, cuyo consumo caliente se intensifica con el frío. Las bombillas son un producto específicamente sudamericano que los proveedores chinos fabrican para exportación a Argentina, Uruguay y Paraguay.
