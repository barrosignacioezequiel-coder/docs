# Cap. 92 — Instrumentos musicales; sus partes y accesorios (Invierno)

> **Capítulo NCM 92** — Instrumentos musicales; sus partes y accesorios | **Temporada:** Invierno (Jun–Ago)

Las **guitarras acústicas para principiantes** son el producto invernal de este capítulo. El invierno es temporada de hobbies indoor y la guitarra es el instrumento más popular para empezar. Pico dual: Día del Padre (junio) como regalo y vacaciones de invierno (julio) como hobby. Formato: guitarras 38-41" con pack completo (funda, púas, correa, afinador).
