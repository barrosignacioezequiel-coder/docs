# Fuentes de Información

Este directorio almacena las URLs y fuentes utilizadas para crear el contenido.

## Formato sugerido

Para cada tema, crear un archivo `.md` con este formato:

```markdown
# [Nombre del tema]

## Fuentes consultadas

| URL | Fecha | Contenido |
|-----|-------|-----------|
| https://ejemplo.com/articulo | 2026-02-08 | Descripción breve |

## Notas
- Observaciones relevantes
- Información que requiere actualización periódica
```

## Archivos por tema

- `proveedores.md` — Fuentes sobre plataformas y proveedores
- `logistica.md` — Fuentes sobre envíos y transporte
- `legal.md` — Fuentes sobre requisitos legales por país
- `costos.md` — Fuentes sobre aranceles e impuestos
